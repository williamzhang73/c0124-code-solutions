# postgres-database-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What is a database schema?
  a collection of tables called schema, schema defines how the database organize data.

- What is a table?
  table is used to organzine data, it has rows and columns, each column refers to one attribute of data, and each row refers to one record.

- What is a row?
  A table has a list of rows, each row represents a record

- What is an attribute and what other names are used to describe them?
  attributes are the features or qualities of a record.
  alternative name of attribute is column.
