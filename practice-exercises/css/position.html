This Is the Menu1

As a UI Engineer you will be responsible for building the next generation of user experiences for our client services. We need a highly motivated and skillful engineer to work with HTML, JavaScript, and CSS and related tooling, frameworks, and libraries. Attention to detail and a dedication to deliver a high-quality, stable delivery is essential. To be successful you are self-motivated; driven to achieve and exceed commitments; participate in pull request (PR) reviews and Agile development. You also need to exude strong collaboration skills, including the ability to mentor and be mentored. Possess a strong passion to work in a growing, energizing environment of innovation. If you want to be part of this amazing team, this position is for you.
